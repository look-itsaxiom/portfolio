About / Summary

I build tools that make people more capable — not tools that replace them. My path wasn't traditional. I started in IT support, fell in love with scripting, and forced my way into software engineering when a director noticed I was automating half my job in PowerShell. That background means I think end-to-end: from infrastructure to interface to user. These days I build across the full stack — React frontends, .NET and Node backends, Azure serverless, Docker deployments — and I do most of it on Windows using Claude Code as my daily driver. That combination led me to build Dream Factory, a Claude Code plugin that turns a single session into a multi-agent software studio, coordinating AI departments across art, engineering, and game development. I think AI is this generation's touchscreen moment — a fundamentally new way to interact with complex systems through natural language. It raises the sea level for everybody's competence. I build AI that covers a person's weaknesses and extends their strengths: a RAG-powered portfolio chat that handles self-promotion I'm bad at, an internal knowledge chatbot deployed where coworkers already work, a card game where an AI arbiter judges player arguments instead of enforcing rigid rules. I ship across a lot of domains — developer tools, web platforms, game engines (Unity, Godot, Phaser), AI pipelines — because interesting problems don't stay in one lane. I have 40+ repos, 65 local projects, and a self-imposed five-project cap to keep myself honest. What I bring: fast ramp-up, strong opinions held loosely, and a bias toward building things people actually use. What I'm looking for: hard problems, good teammates, and work that matters.
